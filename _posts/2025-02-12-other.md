## import mysql data
```
# 1. 源机器导出并压缩
mysqldump -u 用户 -p密码 --databases 数据库名 | gzip > db.sql.gz

# 2. 传输到目标机
scp db.sql.gz 目标用户@目标IP:/path/

# 3. 目标机解压导入
ssh 目标用户@目标IP "gzip -d /path/db.sql.gz && mysql -u 用户 -p密码 < /path/db.sql"
```
